# AI/ML Data Cleaning Project

## Project Overview
This project focuses on cleaning and preparing raw data for analysis.  
The goal is to transform messy data into a structured format ready for machine learning.

## Problem Statement
Raw datasets often contain missing values, duplicates, and inconsistent formatting.  
This project solves that problem by cleaning and standardizing the dataset.

## Dataset
The dataset used contains structured records with multiple columns.  
The data required cleaning before it could be used for analysis.

## Tools Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Jupyter Notebook  

## Data Cleaning Steps
- Removed duplicates  
- Handled missing values  
- Renamed columns  
- Converted data types  
- Removed irrelevant columns  

## Results
The dataset was successfully cleaned and structured.  
It is now ready for analysis or machine learning tasks.

## Insights
- Missing values were found in multiple columns  
- Duplicate rows affected dataset quality  
- Data consistency improved after cleaning  

## Project Structure
data/ → dataset used  
notebooks/ → data cleaning notebooks  
charts/ → visualizations  
requirements.txt → dependencies  

## How to Run
Install dependencies:

pip install -r requirements.txt

Open notebook:

notebooks/data_cleaning.ipynb
